## Update Protocol

- Update this file after substantive learning conversations when doing so will reduce future friction.
- Do not move a concept into `Fast Retrieval` unless the user has successfully explained it back, applied it to a new example, or answered retrieval questions without heavy prompting.
- Do not treat familiarity as mastery.
- Record productive struggle under `Requires Reconstruction`, not as failure.
- When a concept moves categories, add a brief reason or evidence.
